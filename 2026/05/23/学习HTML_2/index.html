HTML基础学习笔记

本笔记根据 HTML/CSS 核心知识点课件及课堂手记整理，涵盖了网页排版、超链接、CSS基础语法与引入方式、选择器、文本样式以及盒子模型等核心内容。

一、 HTML 基础标签与排版

1. 标题与水平线标签

  • 标题标签：<h1> 到 <h6>

  • 水平线标签：<hr>，用于在页面中插入一条水平分割线。

2. 图片标签

  • 标签格式：<img src="图片路径" width="宽度" height="高度">
  • 路径分类：
    • 绝对路径：
      • 绝对磁盘路径：如 D:/xxxx/image.png
      • 绝对网络路径：如 https://xxxx/image.png
    • 相对路径：从当前文件所在目录开始查找。
      • ./ ：当前目录
      • ../ ：上级目录

3. 音频与视频标签

  • 视频标签：<video src="视频路径" controls width="宽度" height="高度">
  • 音频标签：<audio src="音频路径" controls>
  • 核心属性说明：src 指定资源路径；controls 用于显示播放控制面板。

4. 换行与段落标签

  • 换行标签：<br> （单标签）
  • 段落标签：<p> （双标签）
  • 空白字符注意点：在 HTML 中无论输入多少个空格，网页上默认只会显示一个。如果需要连续的空格，可以使用空格占位符：&nbsp;。

5. 文本加粗标签

  • <b> 和 <strong>
  • 注：<strong> 比 <b> 更加具有语义化，强调内容的重要性。

6. 无语义布局标签 <span>

  • 特点：<span> 是一个在开发网页时大量会用到的没有语义的布局标签。
  • 表现：一行可以显示多个（组合行内元素），其宽度和高度默认由内部的文本内容撑开，不支持直接设置宽高。

二、 超链接

  • 标签格式：<a href="url" target="打开方式">链接文本</a>
  • 核心属性：
    • href：指定资源访问的 URL 地址。
    • target：指定在何处打开资源链接。
      • _self：默认值，在当前窗口/标签页打开。
      • _blank：在新空白标签页中打开。
  • 相关CSS修饰：
    • text-decoration：规定添加到文本的修饰。常用 text-decoration: none; 来去除超链接默认的下划线。
    • color：定义超链接文本的颜色。

三、 CSS 基础与引入方式

1. CSS 引入方式

CSS 共有三种主流引入到 HTML 中的方式：

  1. 行内样式（内联样式）：直接在标签的 style 属性中编写。
    • 示例：<h1 style="color: red;">内容</h1>
  2. 内嵌样式：在 HTML 文件的 <head> 标签内使用 <style> 标签包裹 CSS 代码。
    • 示例：<style> h1 { color: blue; } </style>
  3. 外联样式（外部样式表）：将 CSS 代码单独写在 .css 文件中，通过 <link> 标签引入。
    • 示例：<link href="xxx.css" rel="stylesheet">

2. 颜色表示法

在 CSS 中，颜色可以通过以下三种方式表示：

  • 关键字：如 red, green, blue 等。
  • rgb表示法：红(r)、绿(g)、蓝(b)三原色调配，取值范围 0~255。
    • 示例：rgb(255, 0, 0)（红色）, rgb(134, 100, 89)
  • 十六进制法：以 # 开头，后面跟6位或3位十六进制字符。
    • 示例：#ff0000（红色）, #cccccc, #ccc

四、 CSS 选择器与优先级

1. 基础选择器

  • 元素选择器（标签选择器）：通过标签名选择元素。
    • 语法：标签名 { ... }
  • id选择器：通过元素的 id 属性精确定位唯一元素。
    • 语法：#id属性值 { ... }
  • 类选择器（class选择器）：通过元素的 class 属性选择一组元素。
    • 语法：.class属性值 { ... }

2. 选择器优先级

当多个选择器作用于同一个元素时，样式的层叠优先级为：
$$\text{id选择器} > \text{类选择器} > \text{元素选择器}$$

五、 常用 CSS 属性

1. 字体与文本基础

  • color：设置文本的颜色。
  • font-size：字体大小。（注意：记得加 px 单位）。

2. 排版高级属性

  • line-height：设置行高，用于控制行与行之间的垂直间距或实现单行文本垂直居中。
  • text-indent：定义第一个行内容的缩进（常用于首行缩进，如 text-indent: 2em;）。
  • text-align：规定元素中文本的水平对齐方式（如 left, center, right）。

六、 盒子模型 (Box Model)

网页中的每一个元素都可以看作是一个矩形的“盒子”，盒子模型由以下四部分组成：

  1. **content (内容)**：盒子的核心，存放文字、图片等实际内容的区域。
  2. **padding (内边距)**：内容与边框之间的透明区域。
  3. **border (边框)**：围绕在内边距和内容外围的线条。
  4. **margin (外边距)**：控制盒子与盒子之间的外部距离。

📑 盒子模型 (Box Model) 深度补充笔记

网页中的每一个元素都可以看作是一个矩形的“盒子”。在网页布局中，我们通过设置不同的 CSS 属性来调整盒子的大小和间距。

一、 盒子模型的四个核心组成部分

  1. content（内容）：盒子的核心，存放文字、图片等实际内容的区域。
  2. padding（内边距）：内容与边框之间的透明区域。
  3. border（边框）：围绕在内边距和内容外围的线条。
  4. margin（外边距）：控制盒子与盒子之间的外部距离，是透明的。

二、 盒子模型的核心 CSS 属性

为了精确控制盒子，我们需要熟练掌握以下属性：

属性名 功能说明 语法示例及要点
width 设置内容的宽度 width: 200px;
height 设置内容的高度 height: 150px;
border 设置边框的属性 border: 1px solid #000;
(同时设置：线宽 样式 颜色)
padding 设置内边距（内衬） padding: 20px;
margin 设置外边距（间距） margin: 10px;

三、 高级进阶：方位控制与复合写法（重点）

1. 单个方位精准控制

如果你只需要设置某一个方位的边框、内边距或外边距，可以在属性名后面加上 **-位置**（top / bottom / left / right）：

  • 内边距单独控制：padding-top（上）、padding-left（左）、padding-right（右）、padding-bottom（下）。
  • 外边距单独控制：margin-bottom（下外边距，常用于段落间距）。
  • 边框单独控制：border-bottom: 2px solid red;（常用于制作下划线或导航条下边框）。

2. 属性值连写规则（顺时针原则）

  • 4 个值：padding: 10px 20px 30px 40px;
    • ➔ 严格遵循 顺时针原则：上（10px） ➔ 右（20px） ➔ 下（30px） ➔ 左（40px）。

四、 拓展考点：盒子的实际大小计算

在标准样式下，一个盒子在网页上实际占用的空间（总宽高）并不是只有 width 和 height，它包含了：

  • 实际宽度 = width + padding-left + padding-right + border-left-width + border-right-width
  • 实际高度 = height + padding-top + padding-bottom + border-top-width + border-bottom-width

💡 贴心小提示（自动内缩盒子）：
如果你觉得每次加了 padding 或 border 盒子就被“撑大”很麻烦，可以给盒子加上这行 CSS 属性：
box-sizing: border-box; （怪异/IE盒子模型）
这样设置之后，你指定的 width 和 height 就会直接包含 padding 和 border，盒子再也不会被无端撑大了！
