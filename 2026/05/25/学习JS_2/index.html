流程控制、函数与 JS 对象

一、 函数与流程控制

1. 流程控制

包含条件分支（if-else、switch）和循环语句（for、while、do-while）。在 JS 中，以下值在进行布尔条件判断时会被视为 false（称为 Falsy 值）：

false、0、"" (空字符串)、null、undefined、NaN。其余所有值皆为 true。

二、 JS 对象大类

你梳理得很好，提到了：**Array、String、JSON、BOM（浏览器对象模型）、DOM**（文档对象模型）。

3. Array（数组）对象

  • 特点： 类似于 Java 的集合（如 ArrayList），具有长度可变和类型可变（同一个数组里可以同时放数字、字符串、对象等）的特性。

  • 常用属性与方法：

    • length：获取数组长度。
    • forEach()：遍历数组中所有有值的元素。
    • push()：在数组末尾添加一个或多个元素，并返回新长度。
    • splice()：非常强大的方法，可以用来删除、插入或替换数组中的元素。

  • forEach(callback)：遍历数组。

    JavaScript

4. String（字符串）对象

字符串属于原始类型，但 JS 会在其调用方法时自动包装为对象。注意：JS 中的字符串是不可变的，所有方法都会返回一个新字符串，而不会修改原字符串。

  • 常用属性与方法：
    • length：获取字符串长度（注意：在 String 中这是一个属性，不是方法）。
    • charAt(index)：返回指定索引位置的字符。
    • indexOf(searchString)：返回指定子字符串在字符串中首次出现的索引，找不到则返回 -1。
    • trim()：去除字符串两端的空白字符。
    • substring(indexStart, indexEnd)：提取字符串中介于两个指定下标之间的字符（左闭右开 [start, end)）。
